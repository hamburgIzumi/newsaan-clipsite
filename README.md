# newsaan-clipsite 🎬

## 🌐 公開デモサイトURL
👉 **[https://hamburgIzumi.github.io/newsaan-clipsite/](https://hamburgIzumi.github.io/newsaan-clipsite/)**

Twitch配信者 **[newsaan](https://www.twitch.tv/newsaan)** の配信クリップを、外部サイトに遷移することなく、当サイト内で快適に検索・閲覧・再生できる非公式ファンサイト（ギャラリー）です。

本プロジェクトは、Twitch Helix REST API と GitHub Actions を組み合わせた JAMstack 構成を採用しており、大量のクリップデータを自動で全量収集・表示するモダンで快適な設計となっています。

---

## 🚀 主な機能

### 1. サイト内完結型 Twitch クリッププレイヤー
* **シームレスな再生体験**: クリップカードをクリックすると、サイト中央に美しいモーダル（ポップアップ）プレイヤーが起動し、その場で動画が再生されます。
* **ドメイン自動解決 (parent パラメータ)**: Twitchの埋め込みAPI仕様に合わせて、稼働ドメインを JavaScript が `window.location.hostname` から動的に取得して自動バインドします。ローカル開発環境（`localhost`）でも、デプロイ後の本番環境でも、設定を変更することなく完全に動画再生が機能します。
* **快適なUXコントロール**:
  * 動画再生中にモーダルを閉じると、自動的に iframe を破棄して動画の音声再生を完全に停止します。
  * キーボードの `Esc` キー押下や、動画エリア外の背景（オーバーレイ）クリックでもモーダルを閉じることができます。
  * モーダル起動時は背面のスクロールを一時的にロックし、操作性を担保します。

### 2. Twitch Helix API 連携 & 自動更新パイプライン (Python 3)
* **100件制限の完全突破**: 従来のパブリック API が持つ「最大100件までしか遡れない」物理制限を完全に克服。Twitchの公式認証（OAuth2 アプリアクセストークン）を使用し、再帰的なカーソルページネーションループを用いて **何百・何千件でも全量取得可能** な Python スクリプト (`scripts/update_clips.py`) を実装しました。
* **ゲーム名の自動解決**: クリップに紐づく `game_id` を Twitch Helix Games API を通じて一括で逆引きし、フロントエンドに完全なゲームカテゴリ情報を提供します。
* **GitHub Actions による定期実行**: 毎日日本時間 24:00 (UTC 15:00) に GitHub Actions が自動で Python スクリプトを走らせ、最新データ（現在約1,100件）を全量再取得して `data/clips.json` を更新＆自動プッシュします。
* **更新日時の表示**: 静的 JSON からデータ最終更新日時を抽出し、日本時間（JST）で「何年何月何日何時何分時点」のデータかをサイトヘッダーに分かりやすく表示します。

### 3. スマート省略ページネーション (Pagination Engine)
* **レイアウト崩れの防止**: クリップが1,000件を超えてページ数が増加したため、インテリジェントな動的ページネーション省略処理を実装しました。「最初のページ」「現在のページの前後1ページ」「最後のページ」だけを動的に残し、その間の離れたギャップを自動で **`...` (省略記号)** で折りたたむことで、スマホを含むすべての画面サイズで絶対にボタン配置が崩れないレスポンシブデザインを実現しています。

### 4. 青系パステルカラーのプレミアム・デザイン
* **爽やかで優しいライトテーマ**: 重々しいダークトーンから一新し、清潔感と透明感のある **「青系パステルカラー (パステルブルー)」** を基調としたライトテーマへと完全にリファクタリング。
* **高い視認性のタイポグラフィ**: Outfit フォントと Inter フォントを組み合わせ、文字色には目に優しい暗めのスレートグレーを採用しました。
* **美しく透き通るグラスモルフィズム**: 背景がソフトに透けて重なり合うモダンなガラス調パネルと、ソフトな影（ドロップシャドウ）による優れた奥行き表現。

### 5. ユーモラスな「ぷるぷるストッキング顔」ロゴ ＆ 遊び心
* **シュールなロゴ画像**: Twitch配信者 newsaan 様を象徴する、ストッキングを被ったユーモラスな写真 (`img/new1.png`) を、ファビコン（Favicon）およびヘッダーの丸型ロゴに適用しました。
* **ぷるぷるホバーエフェクト**: ロゴ部分にマウスを乗せると、ストッキングの弾力を表すように **「ぷるぷると震えて弾む」ユーモラスなアニメーション (`@keyframes jiggle`)** が発生します。ホバー時にはロゴアイコンの枠線がゴールド（パステルイエロー）に輝き、顔がフワッと拡大する楽しいインタラクションを組み込んでいます。

### 6. 高度な複合検索・フィルター
* **複数キーワード対応**: 全角・半角スペースで区切られた複数の単語による部分一致検索に対応。
* **AND / OR 検索の動的切り替え**: 「すべての単語を含む (AND)」か「いずれかの単語を含む (OR)」かをトグルで瞬時に切り替え可能。
* **単一ゲームカテゴリ選択**: 全クリップデータから存在するゲーム名を動的に抽出し、セレクトボックスで特定のゲームのクリップのみに絞り込めます。
* **日付期間指定フィルター**: カレンダーから開始日と終了日を指定し、特定の期間に作成されたクリップのみを抽出できます。
* **4パターンのソート**: 「閲覧数の多い順・少ない順」「作成日の新しい順・古い順」に対応しています。

## 📁 ディレクトリ構造

```text
newsaan-clipsite/
├── .github/
│   └── workflows/
│       └── update_clips.yml  # GitHub Actions (Python 3 + 依存パッケージ + Secretsバインド)
├── css/
│   └── style.css            # パステルブルー調のライトデザインシステム、ぷるぷるロゴ、省略UI
├── js/
│   └── app.js               # 検索・フィルタリング・モーダル・スマート省略ページングJS
├── data/
│   └── clips.json           # Twitch Helix APIから自動収集した全クリップJSON (データソース)
├── img/
│   ├── new1.png             # ユーモラスなストッキング顔アイコン（ファビコン／ロゴ兼用）
│   └── new2.gif             # ユーモラスな動的参考アセット
├── doc/
│   └── agent_instructions.md # 開発ワークフロー用のエージェント指示書
├── index.html               # メインHTML（ファビコン、 Outfit/Interフォント、Lucide）
└── README.md                # 本説明書 (最新仕様版)
```

## 🛠 動作確認・起動方法

1. 本リポジトリをクローンまたはダウンロードします。
2. フォルダ内の `index.html` を任意のWebブラウザ（Chrome, Edge, Safari等）でダブルクリックして開きます。
3. ローカル環境でも、検索・ソート・動画再生のすべての機能が完全に動作します。
   *(※最新のクリップデータを手元に同期させたい場合は、ターミナルで `git pull origin work` を実行してください)*
