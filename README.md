# newsaan-clipsite 🎬

## 🌐 公開デモサイトURL
👉 **[https://hamburgIzumi.github.io/newsaan-clipsite/](https://hamburgIzumi.github.io/newsaan-clipsite/)**

Twitch配信者 **[newsaan](https://www.twitch.tv/newsaan)** の配信クリップを、外部サイトに遷移することなく、当サイト内で快適に検索・閲覧・再生できる非公式ファンサイト（ギャラリー）です。

---

## 🚀 主な機能

### 1. サイト内完結型 Twitch クリッププレイヤー (Issue #4)
*   **シームレスな再生体験**: クリップカードをクリックすると、サイト中央に美しい半透明のモーダル（ポップアップ）プレイヤーが起動し、その場で動画が再生されます。
*   **ドメイン自動解決 (parent パラメータ)**: Twitchの埋め込みAPI仕様に合わせて、稼働ドメインを JavaScript が `window.location.hostname` から動的に取得・バインドします。ローカル開発環境（`localhost`）でも、デプロイ後の本番環境（GitHub Pages）でも、設定を変更することなく完全に動画再生が機能します。
*   **快適なUXコントロール**:
    *   動画再生中にモーダルを閉じると、自動的に iframe を破棄して動画の音声再生を完全に停止します。
    *   キーボードの `Esc` キー押下や、動画エリア外の背景（オーバーレイ）クリックでもモーダルを閉じることができます。
    *   モーダル起動時は背面のスクロールを一時的にロックし、操作性を担保します。

### 2. 高度な複合検索・フィルター (Issue #3)
*   **複数キーワード対応**: 全角・半角スペースで区切られた複数の単語による絞り込みが可能です。
*   **AND / OR 検索の動的切り替え**: 複数ワード検索時、「すべての単語を含む (AND)」か「いずれかの単語を含む (OR)」かをトグルスイッチでリアルタイムに切り替えることができます。
*   **日付期間指定フィルター**: カレンダーから開始日と終了日を指定し、特定の期間に作成されたクリップのみを抽出できます。

### 3. ソート機能 (Issue #3)
*   閲覧数（Views）の多い順・少ない順
*   作成日（Date）の新しい順・古い順
の計4パターンでの瞬時の並び替えに対応しています。

### 4. プレミアム・デザイン (Issue #1)
*   **モダンなビジュアル**: 暗色（ダークモード）をベースに、Twitchを象徴するパープルのネオングロー効果と、背景が美しく透ける「グラスモルフィズム（ガラス調パネル）」を採用。
*   **レスポンシブ対応**: PCの大画面からスマートフォンの小さな縦画面まで、グリッドレイアウトが自動的に最適化されます。スマホ表示時にはボタンテキストが自動的にアイコン化されるなど、細部まで調整されています。

## 🛠 動作確認・起動方法

1. 本リポジトリをクローンまたはダウンロードします。
2. フォルダ内の `index.html` を任意のWebブラウザ（Chrome, Edge, Safari等）でダブルクリックして開きます。
3. ローカル環境でも、検索・ソート・動画再生のすべての機能が完全に動作します。
