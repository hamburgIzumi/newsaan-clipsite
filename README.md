# newsaan-clipsite 🎬

## 🌐 公開デモサイトURL
👉 **[https://hamburgIzumi.github.io/newsaan-clipsite/](https://hamburgIzumi.github.io/newsaan-clipsite/)**

Twitch配信者 **[newsaan](https://www.twitch.tv/newsaan)** の配信クリップを、外部サイトに遷移することなく、当サイト内で快適に検索・閲覧・再生できる非公式ファンサイト（ギャラリー）です。

本プロジェクトは、Twitch Helix REST API と GitHub Actions を組み合わせた JAMstack 構成を採用しており、大量のクリップデータを自動で全量収集・表示するモダンで快適な設計となっています。

---

## 🚀 主な機能

### 1. サイト内完結型 Twitch クリッププレイヤー
* **シームレスな再生体験**: クリップカードをクリックすると、サイト中央に美しいモーダル（ポップアップ）プレイヤーが起動し、その場で動画が再生されます。
* **ドメイン自動解決 (parent パラメータ)**: Twitchの埋め込みAPI仕様に合わせて、稼働ドメインを JavaScript が `window.location.hostname` から動的に取得して自動バインドします。ローカル環境（`localhost`）でも、Pagesデプロイ後の本番環境でも、設定を変更することなく完全に動画再生が機能します。
* **快適なUXコントロール**:
  * 動画再生中にモーダルを閉じると、自動的に iframe を破棄して動画の音声再生を完全に停止します。
  * キーボードの `Esc` キー押下や、動画エリア外の背景（オーバーレイ）クリックでもモーダルを閉じることができます。
  * モーダル起動時は背面のスクロールを一時的にロックし、操作性を担保します。

### 2. #C2E0FF をベースにした青系パステルカラーデザイン
* **爽やかで優しい極上の配色**: 淡く爽やかな極上のパステルブルー（`#C2E0FF`）をベース色に採用し、清潔感と楽しさを両立したビジュアルを構築。
* **高い視認性のタイポグラフィ**: Outfit フォントと Inter フォントを組み合わせ、文字色には目に優しい暗めのスレートグレーを採用しました。
* **美しく透き通るグラスモルフィズム**: 背景がソフトに透けて重なり合うモダンなガラス調パネルと、ソフトな影（ドロップシャドウ）による優れた奥行き表現。

### 3. グリッド表示 ＆ リスト表示のハイブリッド切り替え
* **選べる表示レイアウト**: クリップ一覧の右上にある切り替えスイッチで、カードが並ぶダイナミックな「グリッド表示」と、1行ずつスッキリ見通せる「リスト表示」を瞬時に切り替えられます。
* **レスポンシブなリストビュー**: リスト表示時、PCでは横長カードとしてタイトルや再生時間・ゲーム名が綺麗に一列に整列し、画面幅の狭いスマホ表示時には自動的に破綻のない縦並び構成へシフトします。表示選択状態は `localStorage` に保存・永続化されます。

### 4. Twitch Helix API 連携 & 自動更新 ＆ リアルタイム件数表示 (Issue #38)
* **100件制限の完全突破**: 従来のパブリック API が持つ「最大100件までしか遡れない」物理制限を完全に克服。Twitchの公式認証（OAuth2 アプリアクセストークン）を使用し、再帰的なカーソルページネーションループを用いて **何百・何千件でも全量取得可能** な Python スクリプト (`scripts/update_clips.py`) を実装しました。
* **「nn本のクリップ」での総数表記**: サイト上に表示される総クリップ数の表記を、ご指定の **「1098本のクリップ」** といった自然な形式へと統一しました。
* **リアルタイムな該当件数バッジの動的拡張**: 検索キーワードやゲームカテゴリ、日付期間などのフィルターが適用されている場合、バッジの表記を動的に **「24本のクリップ (全1098本中)」** のようにリアルタイムで拡張・算出表示するスマートな絞り込み件数可視化機能を実装しました。
* **GitHub Actions による定期実行**: 毎時23分に GitHub Actions が自動で Python スクリプトを走らせ、最新データ（現在約1,100件）を全量再取得して `data/clips.json` を更新＆自動プッシュします。
* **更新日時の表示**: 静的 JSON からデータ最終更新日時を抽出し、日本時間（JST）で「何年何月何日何時何分時点」のデータかをサイトヘッダーに分かりやすく表示します。

### 5. スマート省略ページネーション ＆ ページ指定遷移 (Issue #39, #42)
* **インテリジェントな省略ロジック**: クリップが多ページにわたる際、「最初のページ」「現在のページの前後1ページ」「最後のページ」だけを動的に残し、間のギャップを `...` で折りたたむことで、ボタンのあふれを防ぐ省略ロジックを搭載しています。
* **ページ番号直接指定（ジャンプ）機能**: 複数ページが存在する場合、任意のページ番号をキーボード入力（またはスピン指定）して「移動」ボタンをクリック、あるいは入力欄での **Enter キー押下** により、指定したページへ一瞬で直接ジャンプできるUIを設置しました。
* **安全なバリデーション ＆ スムーズスクロール**: 1から最大ページ数以外の無効な数値が入力された場合はブラウザが警告（アラート）を出して遷移を安全にブロックします。遷移成功時には、クリップ一覧エリアの最上部まで画面が**スムーズに自動スクロール**する高度なUXを適用しています。
* **崩れのない超堅牢レイアウト（Issue #42）**: ページジャンプUI追加に伴う幅不足により、ボタンが潰れてアイコンや文字が縦並びに折り返されてしまう表示崩れを完璧に解消しました。親コンテナへの `flex-wrap: wrap;` の適用によって、画面幅が狭まった場合は自然に折り返して中央配置され、各ボタンやページリストに `flex-shrink: 0;` および `white-space: nowrap;` を適用することで、絶対にテキストが押し潰されない強固なレスポンシブレイアウトを構築しています。

### 6. ユーモラスな「ぷるぷるパンスト顔」ロゴ ＆ 遊び心
* **シュールなロゴ画像**: Twitch配信者 newsaan 様を象徴する、パンストを被ったユーモラスな写真 (`img/new1.png`) を、ファビコン（Favicon）およびヘッダーの丸型ロゴに適用しました。また、ホバーのツールチップ属性を `※パンストを被っています`、ロゴ横のサブテキストを `※パンスト着用中` に更新し、配信者のアイデンティティを忠実に表現しています。
* **ぷるぷるホバーエフェクト**: ロゴ部分にマウスを乗せると、パンストの弾力を表すように **「ぷるぷると震えて弾む」ユーモラスなアニメーション (`@keyframes jiggle`)** が発生します。ホバー時にはロゴアイコンの枠線がゴールド（パステルイエロー）に輝き、顔がフワッと拡大する楽しいインタラクションを組み込んでいます。

### 7. クリップ作成者（Creator）の表示対応
* **コミュニティ感の向上**: 各クリップカードおよび動画再生モーダルに、クリップを実際に作成したTwitchユーザー名（`creator_name`）を動的に表示する機能を追加しました。
* **調和したデザイン**: 
  * クリップカード内では、タイトルのすぐ下にパステルブルーのユーザーアイコンと共にスマートに配置。
  * リストビューでは、横並び構成の間に境界線付きでスッキリとレイアウト。
  * 動画再生モーダル内でも、詳細なメタ情報の1つとしてシームレスに表示されます。

### 8. 高度な複合検索・フィルター
* **複数キーワード対応**: 全角・半角スペースで区切られた複数の単語による部分一致検索に対応。
* **AND / OR 検索の動的切り替え**: 「すべての単語を含む (AND)」か「いずれかの単語を含む (OR)」かをトグルで瞬時に切り替え可能。
* **単一ゲームカテゴリ選択**: 全クリップデータから存在するゲーム名を動的に抽出し、セレクトボックスで特定のゲームのクリップのみに絞り込めます。
* **日付期間指定フィルター**: カレンダーから開始日と終了日を指定し、特定の期間に作成されたクリップのみを抽出できます。
* **4パターンのソート**: 「閲覧数の多い順・少ない順」「作成日の新しい順・古い順」に対応しています。

### 9. 主要モダンブラウザにおける完全な互換性確保 (Issue #40, #41)
* **標準CSS規格プロパティの完全な併記**: 開発のLinterやエディタによる「ベンダープレフィックス特有のプロパティ指定」に対する互換性警告を完全に解消するため、以下のベンダー専用記述に対して、世界基準の標準CSSプロパティを適切に併記しました。
  - 入力欄のスピンボタン非表示: `-webkit-appearance`, `-moz-appearance` に **`appearance: none / textfield`** を併記。
  - 文字のグラデーションクリップ: `-webkit-background-clip: text` に **`background-clip: text`** を併記。
  - テキストの行数制限（三点リーダー）: `-webkit-line-clamp` に **`line-clamp`** をそれぞれ併記。
* これにより、Chrome, Firefox, Safari, Edge などすべての主要ブラウザで将来にわたる安全な描画互換性を保証しています。

## 🛠 動作確認・起動方法

1. 本リポジトリをクローンまたはダウンロードします。
2. フォルダ内の `index.html` を任意のWebブラウザ（Chrome, Edge, Safari等）でダブルクリックして開きます。
3. ローカル環境でも、検索・ソート・動画再生、およびテーマ・表示形式切り替えのすべての機能が完全に動作します。
   *(※最新のクリップデータを手元に同期させたい場合は、ターミナルで `git pull origin work` を実行してください)*
