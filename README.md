# newsaan-clipsite 🎬

## 🌐 公開デモサイトURL
👉 **[https://hamburgIzumi.github.io/newsaan-clipsite/](https://hamburgIzumi.github.io/newsaan-clipsite/)**

---

Twitch配信者 **[newsaan](https://www.twitch.tv/newsaan)** の配信クリップを、外部サイトに遷移することなく、当サイト内で快適に検索・閲覧・再生できる非公式ファンサイト（ギャラリー）です。

本プロジェクトは、Twitch Helix REST API と GitHub Actions を組み合わせた JAMstack 構成を採用しており、大量のクリップデータを自動で全量収集・表示するモダンで快適な設計となっています。

---

## 💡 提供する機能・ソリューション

### 1. サイト内完結型 Twitch クリッププレイヤー
* **その場でのシームレスな再生**: クリップをクリックすると、美しいモーダルプレイヤーが起動し、その場で動画が自動再生されます。
* **スムーズな操作性**: キーボードの `Esc` キーや背景クリックで即座に閉じ、閉じる際には動画の音声再生を自動で完全に停止します。背面のスクロールロックにより、優れた操作性を担保します。

### 2. 高度な複合検索・フィルター ＆ リアルタイム件数表示
* **複数キーワード検索**: スペース区切りの複数単語による部分一致検索に対応（AND / OR 検索を瞬時に切り替え可能）。
* **柔軟な絞り込み**: 動的に生成されるゲーム名での絞り込みや、カレンダーからの期間指定、4パターンの高速ソートに対応。
* **リアルタイム件数バッジ**: フィルター適用時は「**[該当件数] 本のクリップ (全[総数]本中)**」のように、絞り込み結果の規模をリアルタイムで自動更新してバッジ表示します。

### 3. レスポンシブ ＆ ページジャンプ機能付きページネーション
* **ページ直接遷移（ジャンプ）**: 任意のページ番号を入力し、「移動」をクリックするか **Enter キー** を押すだけで、そのページへ一瞬で直接ジャンプできます（無効な数値の入力は自動ブロック）。
* **スムーズスクロール**: 遷移完了時はクリップ一覧の最上部へスムーズに自動スクロールします。
* **絶対に崩れないレスポンシブ**: 横幅が狭まった場合は自然に折り返して中央配置され、要素が押し潰されて文字やアイコンが縦方向に崩れるのを完全に防止する堅牢なレイアウトを適用しています。

### 4. パステルブルー（#C2E0FF）ベースの極上デザイン
* **爽やかで優しい極上の配色**: 淡く優しいパステルブルー（`#C2E0FF`）をベースに、背景がソフトに透けて重なり合うガラス調のグラスモルフィズム表現を採用。
* **表示スタイルの切り替え**: 「グリッド表示（カード）」と、サムネイルを非表示にした極めてコンパクトな「リスト表示（1行）」を瞬時に切り替え可能です。
* **マルチブラウザ互換性の完全確保**: 標準のCSS規格プロパティの完全な併記（`appearance`、`background-clip`、`line-clamp`）により、すべてのモダンブラウザで警告フリーで完璧な描画を保証します。

### 5. クリップ作成者（Creator）表示 ＆ 遊び心のある演出
* **作成者名（Creator）表示**: クリップカード、リストビュー、動画モーダルのすべてに、クリップ作成ユーザー名をスマートに表示します。
* **ぷるぷるパンスト顔ロゴ**: 配信者を象徴するパンスト顔ロゴ（`img/new1.png`）を適用。マウスホバーでパンストの弾力を表すように「ぷるぷる震えて弾む」アニメーションや、ゴールドに輝くエフェクトを搭載しています。

### 6. データ自動収集パイプライン
* **API制限の突破**: Twitch Helix APIの100件制限を克服し、Pythonスクリプトで何千件ものデータを全量自動収集。
* **定期全自動更新**: 毎時 GitHub Actions が自動でスクリプトを実行し、最新データを自動取得して `clips.json` を更新。サイト上に日本時間でのデータ最終更新日時を分かりやすく表示します。

## 📁 ディレクトリ構造

```text
newsaan-clipsite/
├── .agent/
│   ├── skills/
│   │   └── RULES.md          # 開発規律・レビュー規律（レイアウトテスト義務化条項含む）
│   └── instruction.md        # ローカル管理用アサイン指示書 (※コミット対象外)
├── .github/
│   └── workflows/
│       └── update_clips.yml  # GitHub Actions 自動データ更新ワークフロー
├── css/
│   └── style.css            # #C2E0FFパステルベース配色、ダークモード、横長一行リスト、ぷるぷるロゴ、ジャンプUI
├── js/
│   └── app.js               # 表示切替、複合検索、リアルタイムバッジ件数更新、ページジャンプ処理JS
├── data/
│   └── clips.json           # Twitch Helix APIから自動収集した全クリップJSON (データソース)
├── img/
│   ├── new1.png             # ユーモラスなパンスト顔アイコン（ファビコン／ロゴ兼用）
│   └── new2.gif             # ユーモラスな動的参考アセット
├── index.html               # メインHTML（検索・フィルター、ページネーション＆ジャンプUI）
└── README.md                # 本説明書 (最新仕様・ソリューション簡潔版)
```

## 🛠 動作確認・起動方法

1. 本リポジトリをクローンまたはダウンロードします。
2. フォルダ内の `index.html` を任意のWebブラウザ（Chrome, Edge, Safari等）でダブルクリックして開きます。
3. ローカル環境でも、検索・ソート・動画再生、およびテーマ・表示形式切り替えのすべての機能が完全に動作します。
   *(※最新のクリップデータを手元に同期させたい場合は、ターミナルで `git pull origin work` を実行してください)*
